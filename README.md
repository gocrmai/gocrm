# GOPOS CRM System

創域科技有限公司客戶關係管理系統

## 系統功能

- 客戶資料管理
- 工作紀錄（任務）系統
- 硬件設備設置
- 維修單 PDF 生成
- 電子簽名
- PWA 支援（可安裝到桌面）

## 技術架構

- Python 3 + Django 4
- Bootstrap 5
- SQLite / PostgreSQL
- 部署平台：Zeabur

## 開發環境

```bash
# 創建虛擬環境
python3 -m venv venv

# 激活虛擬環境
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# 安裝依賴
pip install -r requirements.txt

# 運行服務器
python manage.py runserver
```

## 部署到 Zeabur

1. 推送代碼到 GitHub
2. 在 Zeabur 連接 GitHub 倉庫
3. Zeabur 自動部署
