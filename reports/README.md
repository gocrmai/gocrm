# GOPOS Reports - 餐飲集團雲端報表系統 (示範版)

## 系統功能

- 多店鋪即時營業報表
- 日期範圍選擇
- 銷售趨勢圖表
- 店鋪表現排名
- 手機優先 PWA 設計

## 技術架構

- Python 3 + Django 4
- Chart.js 圖表
- Bootstrap 5 響應式設計
- SQLite 示範數據庫
- Zeabur 部署
